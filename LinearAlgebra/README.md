# Linear Algebra Vector Library

Проект реализует математический вектор с базовыми операциями линейной алгебры.

## Структура проекта

- **LinearAlgebra** - библиотека классов, содержащая:
  - `IMathVector` - интерфейс математического вектора
  - `MathVector` - реализация математического вектора
- **VectorDemo** - консольное приложение для демонстрации функциональности

## Функциональность

- Базовые операции с векторами (сложение, вычитание, умножение, деление)
- Скалярное произведение
- Вычисление длины вектора
- Вычисление расстояния между векторами
- Перегрузка операторов
- Обработка исключений

## Использование

```csharp
var vector1 = new MathVector(1, 2, 3);
var vector2 = new MathVector(4, 5, 6);
var sum = vector1 + vector2;
var dotProduct = vector1.ScalarMultiply(vector2);
